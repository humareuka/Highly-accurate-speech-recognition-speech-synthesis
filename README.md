# モデルについて

**モデル名	容量	必要な VRAM**
- tiny	< 400MB	~ 1 GB
- base	< 400MB	~ 1 GB
- small	461MB	~ 2 GB
- medium	1.42GB	~ 5 GB
- large	2.87GB	10 GB
